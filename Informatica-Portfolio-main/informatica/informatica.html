Informatica

Waarom heb ik informatica gekozen?

Van jongs af aan heb ik altijd al interesse gehad in alles rondom techniek. Zo vond ik het altijd al erg leuk om me bezig te houden met elektronica en andere vormen van techniek. Echter, er is één vorm van techniek waar ik van nature de meeste interesse in heb, namelijk computertechniek.

Ik interesseer me altijd heel erg in hoe computers werken, zowel op softwaregebied als op het gebied van hardware. Daarom vind ik het ook erg leuk om de technische vooruitgang op het gebied van hardware in de gaten te houden en doe ik daar ook graag onderzoek naar. Op softwaregebied heb ik daar ook altijd al erg interesse in gehad, ik vond het erg leuk om daar dingen over te leren.

Op een dag deed zich het volgende voor: ik was met mijn moeder in de bibliotheek om een boekje te halen om te lezen en op eens kwam ik het volgende tegen: ‘Games ontwerpen voor kids’ van de boekenserie voor dummies. Dit leek me interessant. Mijn vader hielp me thuis om de benodigde software hiervoor te installeren en ik ging hiermee aan de slag. Dit vond ik heel erg leuk en daarom deed ik het best veel. Maar op den duur liep ik tegen een barrière aan, de beperkingen van scratch. Daarom raadde mijn vader me aan om een boek te kopen om te leren programmeren in C#.

Hier ging ik enthousiast mee aan de slag en dat vond ik echt ontzettend leuk en ook vandaag de dag vind ik het ontzettend leuk om te programmeren. Tegenwoordig is het zo dat ik naast C# ook Java ken, Java was niet zo moeilijk om te leren, aangezien het ontzettend op C# lijkt. Bovendien is het zo dat als je het principe achter programmeren snapt, het makkelijker is om een andere taal te leren. In ieder geval, Java gebruik ik voor het schrijven van plugins voor de game Minecraft. C# gebruik ik voor alle andere projecten.
